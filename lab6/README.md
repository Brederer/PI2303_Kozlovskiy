# Лабораторная работа 6

## Формы для ввода пользовательской информации

Цель работы: ознакомиться с основными виджетами для ввода и редактирования
пользовательской информации, а также с методами валидации форм.

## Выполнено

- Создан отдельный Flutter-проект в папке `lab6`.
- Реализован калькулятор площади.
- Добавлены поля ввода ширины и высоты.
- Реализована проверка пустых полей.
- Реализована проверка некорректного числового ввода.
- При успешном вводе выводится результат в формате `S = ширина * высота = результат`.
- После вычисления показывается всплывающая подсказка об успешном вычислении.
