# Лабораторная работа 8

## Получение данных с использованием бесплатных REST API

Цель работы: разработать приложение, которое получает данные через интернет
и отображает их списком.

## Выполнено

- Создан отдельный Flutter-проект в папке `lab8`.
- Добавлены зависимости `http` и `intl`.
- Реализована фотогалерея на основе данных `jsonplaceholder`.
- Реализована новостная лента КубГАУ.
- Данные загружаются через REST API.
- JSON-ответы преобразуются в модели `Photo` и `NewsItem`.
- Фотографии отображаются через `GridView`.
- Новости отображаются карточками через `Card`.
- Виджеты разделены по файлам для удобной демонстрации кода.

## Запуск

```bash
flutter pub get
flutter run
```
