# Лабораторная работа №10  
**Наследование, интерфейсы, паттерны проектирования**

## Цель работы
Освоить принципы объектно-ориентированного программирования:
- наследование
- интерфейсы
- декомпозиция системы
- применение паттернов проектирования

##  Используемые концепции

- Абстракция (ICoffee)
- Наследование (реализации кофе)
- Перечисления (CoffeeType)
- Паттерн **Factory Method**
- Избавление от антипаттерна **God Object**

## Описание работы

Кофемашина была переработана с использованием ООП:

- Каждый тип кофе вынесен в отдельный класс
- Используется интерфейс `ICoffee`
- Создание объектов реализовано через `CoffeeFactory`
- Ресурсы вынесены в отдельный класс `Resources`

## Реализованные паттерны

### Factory Method
Позволяет создавать объекты кофе без привязки к конкретным классам.
