# Лабораторная работа 3

## Первый проект на Flutter

Цель работы: создать стартовый проект Flutter, рассмотреть структуру
приложения и изменить стандартный счетчик по заданию.

## Выполнено

- Название приложения и заголовок заменены на «Инкремент».
- Тема приложения изменена на другой цвет.
- Текст над счетчиком заменен на «Значение инкремента:».
- Кнопка из нижнего правого угла удалена.
- В центр добавлены кнопки с иконками «-» и «+».
- Добавлена кнопка «Сбросить» для обнуления счетчика.

## Запуск

```bash
flutter pub get
flutter run
```
