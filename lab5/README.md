# Лабораторная работа 5

## Работа со списками

Цель работы: ознакомиться с виджетами для отображения списка информации
и создать несколько страниц приложения со списками.

## Выполнено

- Создан отдельный Flutter-проект в папке `lab5`.
- Реализован простой список из трех строк.
- Реализован бесконечный список с информацией по номеру строки.
- Реализован бесконечный список со степенями числа 2.
- Каждый виджет списка вынесен в отдельный файл:
  - `simple_list.dart`;
  - `infinity_list.dart`;
  - `infinity_math_list.dart`.
- Для переключения между списками используется `TabBar`.

## Запуск

```bash
flutter pub get
flutter run
```
