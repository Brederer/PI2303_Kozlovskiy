# Лабораторная работа №11

## Тема
Асинхронность

## Цель
Освоение механизмов async/await и Future.

## Реализация

Добавлены асинхронные процессы:
- нагрев воды (3 сек)
- заваривание кофе (5 сек)
- взбивание молока (5 сек)
- смешивание (3 сек)

Параллельные процессы реализованы через Future.wait()
